# ✨ About QuickChat

**QuickChat** is a modern full-stack real-time messaging application designed for fast and seamless communication.

It provides instant messaging, secure authentication, responsive UI, and real-time updates using WebSockets. Built with the MERN stack and Socket.IO, the application delivers a smooth chatting experience across desktop and mobile devices.

| Feature | Description |
|---|---|
| ⚡ Real-Time Messaging | Instant communication using Socket.IO |

# ⚙️ Installation

## Clone the Repository

```bash
git clone https://github.com/your-username/quickchat.git
cd quickchat
```

---

## Install Frontend Dependencies

```bash
cd client
npm install
```

---

## Install Backend Dependencies

```bash
cd ../server
npm install
```

---

# 🔧 Environment Variables

## Backend (`server/.env`)

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
CLIENT_URL=http://localhost:5173
```

---

## Frontend (`client/.env`)

```env
VITE_SERVER_URL=http://localhost:5000
```

---

# ▶️ Running the Application

## Start Backend

```bash
cd server
npm start
```

---

## Start Frontend

```bash
cd client
npm run dev
```

Now open:

```txt
http://localhost:5173
```
